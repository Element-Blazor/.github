# 欢迎来到 Element-Blazor 组织 👋

**Element-Blazor** 是一个专注于 Blazor 生态系统的开源组织，致力于为 .NET 开发者提供高质量的 UI 组件库和社区工具。

---

## 🚀 主要项目

### [Element-Blazor](https://github.com/Element-Blazor/Element-Blazor)
[![NuGet Downloads](https://img.shields.io/nuget/dt/Element.svg)](https://www.nuget.org/packages/Element/)
[![GitHub Stars](https://img.shields.io/github/stars/Element-Blazor/Element-Blazor?style=social)](https://github.com/Element-Blazor/Element-Blazor)

基于 [Element UI](https://element.eleme.cn/#/zh-CN/component/layout) 和 Blazor WebAssembly 的 Web UI 组件库。

- API 设计模仿 Element，上手成本低
- 直接复用 Element 的 CSS 样式与 HTML 结构
- 同时支持 **WebAssembly** 和 **Server-Side Rendering（SSR）**
- 在线演示：[element-blazor.github.io](https://element-blazor.github.io/)

### [BlazorCommunity](https://github.com/Element-Blazor/BlazorCommunity)
使用 Element-Blazor 构建的社区系统，涵盖管理后台与前台展示，支持客户端与服务端渲染两种模式。

---

## 🛠️ 技术栈

| 技术 | 描述 |
|------|------|
| [Blazor](https://dotnet.microsoft.com/apps/aspnet/web-apps/blazor) | Microsoft .NET 的 WebAssembly / Server 框架 |
| [Element UI](https://element.eleme.cn/) | 饿了么前端团队出品的 Vue.js UI 组件库（样式来源）|
| C# / .NET | 主要开发语言 |

---

## 📦 快速开始

```bash
dotnet add package Element
```

详细使用文档请参考 [Element-Blazor 仓库](https://github.com/Element-Blazor/Element-Blazor)。
